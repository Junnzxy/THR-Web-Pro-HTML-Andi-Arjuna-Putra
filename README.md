# THR-Web-Pro-HTML-Andi-Arjuna-Putra

Responsive Static Websites: Tugas Jadwal & Form Pendaftaran

Ni hao! Repositori ini berisi dua tugas dasar pemrograman web untuk mata kuliah Web Programing. Fokus utama di sini adalah memaksimalkan penggunaan HTML murni tanpa bantuan CSS atau JavaScript.

Penjelasan Tugas
Tugas ini terdiri dari dua bagian utama yang melatih ketelitian dalam menyusun tata letak (layouting) menggunakan struktur tabel:
- Jadwal Kuliah: Mereplikasi desain jadwal kuliah yang presisi secara visual, mulai dari penggabungan jam (rowspan) hingga pewarnaan sel yang akurat.
- Form Pendaftaran OSIS: Membuat formulir input data fungsional dengan elemen lengkap (text, radio, checkbox, dropdown, hingga file upload) dengan tampilan yang rapi.

Metode & Teknologi yang Digunakan
- Karena aturannya No CSS & No JS, saya menggunakan pendekatan Old-School Web Development:
- Layouting via Tables: Semua penempatan elemen diatur menggunakan tag table, tr, dan td. Ini adalah teknik paling dasar untuk menjaga struktur elemen tetap sejajar.
- Inline Styling Atributes: Untuk pewarnaan dan perataan, saya menggunakan atribut bawaan HTML seperti bgcolor, align, valign, cellpadding, dan cellspacing.
- Typography: Menggunakan tag <font> dan inline style sederhana untuk menentukan jenis font (seperti Times New Roman) agar identik dengan permintaan desain.
- Form Validation Dasar: Menggunakan atribut bawaan HTML5 seperti placeholder dan required untuk memastikan fungsionalitas formulir berjalan baik.

Isi Tugas
1. Jadwal Kuliah
Akurasi Warna: Menggunakan kode hex warna yang spesifik agar mata kuliah tidak tertukar.
Struktur Waktu: Penggunaan rowspan yang teliti untuk mata kuliah yang durasinya lebih dari satu jam.
Responsive Width: Tabel diatur dengan lebar statis agar tetap konsisten saat dibuka di berbagai layar.

2. Form Pendaftaran OSIS
Komponen Lengkap: Menyertakan semua tipe input yang dibutuhkan dalam sebuah sistem pendaftaran.
Visual Teber: Menebalkan label menggunakan tag <b> agar hierarki informasinya jelas.
Header Design: Membuat header yang eye-catching hanya dengan tabel berwarna biru dan teks putih.

Cara Menjalankan
- Nggak perlu setup yang ribet, cukup ikuti langkah simpel ini:
- Clone repositori ini ke perangkat kamu.
- Cari file jadwal.html atau form.html.
- Klik kanan pada file tersebut, lalu pilih Open with dan pilih browser favorit kamu (Chrome, Edge, atau Firefox).

Boom! Tugas langsung tampil di layar.
